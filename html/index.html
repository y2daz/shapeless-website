Shapeless

The Deformation and Fluid Dynamics Plug-In

Shapeless is a plugin for the Unity game engine to simulate 3-dimensional mesh deformation and fluid dynamics, in real time. The system uses the spring-mass model to implement deformable solids.


This method employs a physically based model of objects to produce accurate deformation.


Fluids are simulated through a metaball approximation with the Unity game engine, using Unity circle collider and sprites. Properties like density, viscosity and surface tension have been implemented.


Transparency for the metaballs and the other visual fluid properties have been implemented using specialised shaders.


This system was developed to aid independent Unity developers who are unable to produce fluid systems and deformable object systems in their games, as well as the open source community in an industry where most technologies are closed-source and proprietary.
